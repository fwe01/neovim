# Windows

1. Install scoop
1. Install packages

```
scoop install fzf
scoop install gcc
scoop install ripgrep

scoop bucket add extras
scoop install lazygit
```

1. Untuk lazy git harus dipancing dulu jalan manual di luar nvim pakai `lazygit`

# Install  LSP

1. `:Mason`
1. Pilih tab `LSP`
1. Kalau dah tau namanya, `:MasonInstall {namanya}`
